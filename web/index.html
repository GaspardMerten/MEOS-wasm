The binary

3.6 MiB

WebAssembly, wasm64 / MEMORY64=1, -O2. Statically linked: GEOS, PROJ (with SQLite), JSON-C, GSL.

Bundled data

9.7 MiB

Preloaded proj.db for coordinate reprojection and the IANA tz database for timezone-aware timestamp parsing. Both live on the wasm MEMFS; no network round-trips.

Patches

0

Applied to upstream MEOS. The wasm64 ABI is LP64, so all of Postgres's sizeof(long)==8 assumptions ride along for free.

Architecture

LP64

sizeof(void*) = 8, sizeof(long) = 8, size_t = 8. Pointers cross the FFI boundary as JS BigInt. The heap is addressed by i64 memory instructions.
